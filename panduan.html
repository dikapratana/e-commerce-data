Dokumen ini menjelaskan langkah paling sederhana untuk menjalankan aplikasi dari awal sampai bisa digunakan.

1. Persiapan Awal

Pastikan sudah terinstall di komputer:
  • Node.js versi minimal 24
  • npm (sudah termasuk di Node.js)
Cek dengan perintah:
node -v npm -v

2. Install Dependency

Masuk ke folder project lalu jalankan:
npm install

3. Menjalankan Backend (JSON Server)

Backend WAJIB dijalankan lebih dulu sebelum membuka aplikasi.
Jalankan perintah:
npm run server

Jika berhasil, API dapat diakses di: http://localhost:3000

4. Menjalankan Frontend

Buka terminal baru lalu jalankan:
npm run dev

Buka di browser: http://localhost:5173

5. Cara Penggunaan Singkat

  1. Buka aplikasi di browser
  2. Login ke sistem
  3. Masuk menu Customer untuk kelola pelanggan
  4. Masuk menu Transaksi untuk pembelian paket

6. Jika Terjadi Masalah

Data tidak muncul?
  • Pastikan npm run server masih berjalan
  • Cek bisa buka http://localhost:3000
Port bentrok?
  • Tutup aplikasi lain yang memakai port 3000 atau 5173

7. Cara Menghentikan Aplikasi

Tekan pada terminal:
CTRL + C
